#### Koala 线上直接可运行版本：

###### 适合环境：Linux x64位系统 （内核版本 >= 2.6.28）

###### 编译环境：CentOS 7 / x64
###### Go版本：1.7


##### 下载运行

```
cd ~
wget https://raw.githubusercontent.com/heiyeluren/koala/main/output/koala_online.tar.gz

tar zxvf koala_online.tar.gz
cd koala_online
sh +x bin/*
bin/koala_ctl start

```
<br />

说明：本程序必须在Linux环境下才能工作，如果命令或者执行出错请随时调整，或者查看相关环境是否满足。
